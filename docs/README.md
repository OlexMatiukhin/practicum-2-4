# Architecture Diagrams

This folder contains C4 Model diagrams for the Nimble Modulith solution.

Files:

- `c4-level-1-system-context.puml` - System Context diagram
- `c4-level-2-container.puml` - Container diagram
- `c4-level-3-components.puml` - Component diagram for the Web Host modular monolith

The diagrams are written in PlantUML with C4-PlantUML syntax. They can be rendered with:

- PlantUML plugin in Visual Studio Code or Rider
- PlantUML CLI
- PlantUML server

Example:

```powershell
plantuml docs/*.puml
```
